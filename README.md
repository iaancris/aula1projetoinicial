# aula python
Código desenvolvido da ilustração da aula.
O código apura as seguintes informações: Nome
do cliente, valor da compra e aplica um desconto,
caso o valor ultrapasse 100 reais.
é utilizado o comando while para repetir o processo,
e ao final, é criado uma variavel com o comando input
para que o usuário diga se deseja ou não realizar outro
atendimento.
caso sim, o código reinicia pedindo as informações do
novo atendimento. caso não, é mostrado a informação:
"programa encerrado", através do comando print().
